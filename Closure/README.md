# Обработчик сверок подрядчиков (ДСП)

Автоматический ETL‑инструмент для разделения закупок на основную часть и ДСП в отчётах подрядчиков.

## Возможности
- Загрузка Excel-файла сверки (формат как в примере).
- Автоматическое определение начала ДСП по двум правилам:
  1. Первое дробное значение в столбце `Сумма без НДС`.
  2. Скачок суммы более чем в 10 раз между соседними строками.
- Очистка от мусора: нулевые строки, `olv_campaign`.
- Группировка по кампаниям с расчётом показов (тыс.) и суммы без НДС.
- Вывод двух таблиц: «Основное» и «ДСП».
- Экспорт результата в Excel с двумя листами.
